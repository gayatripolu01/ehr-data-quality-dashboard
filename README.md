# EHR Data Quality Monitoring Dashboard

## Objective
This project evaluates the quality of Electronic Health Record (EHR) data used in clinical research.

## Overview
EHR data often contains missing, inconsistent, or outdated information. This project analyzes data quality using four key dimensions:
- Completeness
- Accuracy
- Timeliness
- Consistency

## Dataset
- 11,500 patient records
- Clinical and treatment data

## Results
- Overall Quality Score: 92.5%
- Completeness: 100%
- Accuracy: 100%
- Timeliness: 70%
- Consistency: 100%

## Outcome
- Improved understanding of EHR data quality
- Supports better clinical research decision-making
